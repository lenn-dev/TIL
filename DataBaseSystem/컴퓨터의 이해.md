## 1. 처리장치
#### 중앙처리장치 CPU (Central Processing Unit)  : 
컴퓨터 시스템 통제, 프로그램 연산 실행 처리하는 가장 핵심적인 컴퓨터의 제어장치,혹은 칩
처리 실행하는 전자회로, 레지스터가 들어있음
정보 입력, 기억, 프로그램 명령어(기계어) 해석, 연산, 데이터 이동, 시스템 입출력 제어

- __프로세서 레지스터__ : 처리할 명령어 저장
- __산술논리연산장치(ALU)__ : 가감승제, 논리명령(논리,비교,Shift) 처리하는 전자회로
- __제어장치(Control unit)__ : 컴퓨터시스템 전체를 지시감독하여 조정. 입력과 출력제어, 연산제어, 데이터 이동
  
## 2. 수치데이터의 표현
#### 1. 데이터의 저장
컴퓨터 시스템의 전자회로는 전자신호인 임펄스(impulse)의 유무 감지
데이터 기본단위는 비트(bit: binary digit)
bit 가 최소 단위이지만 최근 저장장치용량 증가로 바이트(Byte)단위를 주로 이용
이진수  1 ( 전류 on 상태 ) 또는 0 ( 전류 off 상태 )으로 저장,

1 byte = 8 bit = 2^8 만큼 표현됨(?)
1 word = 4 byte = 32 bit(프로세서의 레지스터 크기를 의미하고 과거엔 32비트, 현재는 대부분 64비트)

#### 2. 수치 데이터의 표현
컴퓨터 메모리, 저장장치, 네트워크, 하드디스크 등 디지털시스템에 모든 데이터는 2진법(Binary)으로 데이터를 저장하고 처리.
(10진법) 253 = 2*100 + 5*10 + 3*1 
            = 2*10^2 + 5*10^1 + 3*10^0  
(2진법) 2로 __0이 남을 때까지__ 나눠서 나머지 밑에서부터 위로 기록
253 2진법으로 표현하면 11111101 
= 1*2^7 + 1*2^6 + 1*2^5 + 1*2^4 + 1*2^3 + 1*2^2 + 0*2^1 + 1*2^0 
= 128 + 64 + 32 + 16 + 8 + 4 + 0 + 1

```
#include <stdio.h>

int main()
{
    int decimal = 13; // 변환할 10진수 13을 변수 decimal에 저장
    int binary[20] = { 0, }; // 2진수를 저장할 배열 binary를 선언한 뒤 0으로 초기화

    int position = 0;
    while (1)
    {
        binary[position] = decimal % 2;    // 2로 나누었을 때 나머지를 배열에 저장
        decimal = decimal / 2;             // 2로 나눈 몫을 저장해서 while loop 돌며 계속 나눠주기

        position++;    // 나머지 저장 후 다음 자리 저장하도록 값 증가

        if (decimal == 0)    // 몫이 0이 되면 반복을 끝냄
            break;
    }

    // index 0부터 저장했으니 배열의 요소를 역순으로 출력
    //position - 1부터 0까지 감소
    for (int i = position - 1; i >= 0; i--)
    {
        printf("%d", binary[i]);
    }

    printf("\n");

    return 0;
}
```

## 4. 주기억장치 주소와 데이터 저장
#### 1. 주기억 장치 주소와 데이터 저장
- 기억 용량 단위 : bit, byte, word
- 1Byte = 8 bits ( 한 문자 표현, 유니코드 제외)
- 1KB = 2^10 Bytes (1024 Bytes)
- 1MB = 2^20 Bytes (1024 KB)
- 1GB = 2^30 Bytes (1024 MB)
- 1TB = 2^40 Bytes (1024 GB)
- 1PB = 2^50 Bytes (1024 TB)

- 주기억장치의 각 바이트는 단일 주소를 갖는다.
- 숫자, 영문자, 특수문자가 입력장치에서 읽혀 주기억장치로 보내지면 비트형태로 바뀌어 기억됨
- 한 번 수록된 데이터는 내용이 바뀔때까지 남아있음
- 고급언어로 작성된 프로그램 => 기계어로 번역, 주기억장치저장, 처리기 전자회로가 해석/실행

#### 2. 데이터의 처리방식과 입출력
